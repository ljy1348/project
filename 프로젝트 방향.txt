1. 예약

-- 운항정보 불러오기
-- 운항정보 선택 후 결제
-- 결제 후 예약번호/결제ID 자동생성
-- 페이지 완성 후 회원/비회원 진행 버튼으로 권한별 페이지 설정

2. 예약 조회

-- 기존 유저ID 검색 기능 삭제
-- 예약 조회 선택 시 로그인/비로그인 예약 조회 창 표시
-- 로그인 시 운항정보 불러와서 모든 정보 표시
-- 비로그인시 예약번호, 탑승일,출발지,도착지, 예약자 영문 성 입력 후 조회


3. 체크인

-- 체크인 시 운항정보 조회X, 예약번호로만 조회(운항정보 조인 시 4만5천건 검색해서 조회 오래걸림)
-- 체크인 조회 예약번호로 조회 시 예약번호별 체크인 상태 Y/N 값 표시
-- 유료 좌석 구매 시 구매페이지로 이동, 결제 정보 조회X(좌석 번호별로 가격 지정, 상품코드 등록 X)
-- 페이지 완성 후 체크인 버튼 클릭 시 로그인/비로그인 체크인 등록 표시

4. 수화물


-- 체크인 ID 로 조회 및 신청
-- 체크인 ID 로 조회 시 수화물 등록 여부 Y/N 값 표시
-- 체크인 ID 당 한 건 등록 가능, 및 수정, 삭제(등록 취소) 기능
-- 페이지 완성 후 수화물 버튼 클릭 시 로그인/비로그인 수화물등록 표시


5. 마이페이지 - 결제 정보

-- 운항정보 조회 X(조인 발생으로 성능저하)
-- 결제 ID 별 상품명, 가격, 인증번호, 예약 번호만 표시
-- 운항 ID 는 예약 조회 페이지에서 조회 할 수 있도록 페이지 이동 버튼 기능

6. 마이페이지 - 내 정보

-- 기존 테이블처럼 전체 조회된 후 수정 버튼 클릭 시 수정 가능하게
-- 회원 정보 수정만 가능하게(성별, 생년월일등 변경 X)

7. 고객센터

-- 게시글 ID 자동생성
-- 게시판별 카테고리 지정해서 코드번호 부여, 코드번호별 정리
-- 답변 여부 Y,N 조회 기능 - N 일 때는 답변자 표시 X / Y 일 때 답변자 표시
-- 페이지 완성 후 필수 로그인 후 이용가능하도록 페이지 추가 
-- 페이지 완성 후 내 문의함 버튼 추가
